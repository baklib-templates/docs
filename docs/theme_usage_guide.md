# 模板使用说明 — Docs

## 1. 模板概览 (Theme Overview)

### 模板名称

**Docs**（内部标识：`docs`，版本 **1.2.1**，语法版本 **v2**，范围 **Wiki**）

### 模板描述

打造用户愿意阅读的文档体验：结构清晰、检索迅捷，并可接入 AI 问答。将知识库呈现为产品文档，无需从零自建文档站。

本主题为 Baklib **Wiki** 纯文档型主题，围绕 `templates/` 下的首页变体与统一内容页模板构建：三套首页布局（Class经典布局 / Tree文档布局 / Card文档布局，对应 `index.liquid` / `index.tree.liquid` / `index.card.liquid`，`template_style` 为空 / `tree` / `card`）共用一致的配置字段 ID，便于在后台切换样式时保留数据；`page.liquid` 根据站点首页所选变体自动复用同族侧栏、页内目录与 AI 侧栏等外壳。配套 `snippets/index/` 按变体组织首页与正文片段，并提供 AI 聊天、搜索弹窗、页面反馈等共用能力。全局设置涵盖品牌 Logo、页脚信息、页面元数据展示、微信分享与自定义 HTML 注入等。

### 适用场景

- **产品文档 / 技术文档**：侧栏导航、页内目录、Copy page、分享到 LLM 等面向研发与产品的阅读场景
- **API 指南与开发者文档**：首页热门搜索、快速指引与频道入门卡片，帮助用户快速定位接口与规范
- **企业内部知识库（Wiki 形态）**：栏目页（子页面列表）、推荐内容与首页快速指引，适合规范、最佳实践类内容聚合
- **需要 AI 辅助阅读与检索的文档站**：全站搜索弹窗、AI 问答侧栏、可配置 AI 回答结束语（需租户开通相应能力）

本主题**不**面向通用 FAQ / 工单式帮助中心门户；若站点以短问答、工单为主，应选择其他范围的主题。

### 主要特性

- **三套首页布局**：Class经典布局（渐变 Hero + 入门卡片）、Card文档布局（左侧栏集成 Logo / 搜索 / 全站导航）、Tree文档布局（Hero + 快速指引 + 推荐列表 + 底部反馈横幅）；后台配置字段 ID 一致，便于变体间迁移
- **首页二选一逻辑**：可指定「首页展示的站内页面」在 `/` 渲染指定页面内容，或留空使用当前变体默认首页区块（主标题、热门搜索、快速指引、推荐阅读等）
- **统一正文外壳**：内容页 `page.liquid` 随站点首页 `template_style`（空 / `tree` / `card`）渲染对应布局，保证首页与内页体验一致
- **AI 与搜索**：AI 侧栏、搜索模态框、热门搜索关键词、AI 回答结束语可配置
- **页面工具与反馈**：页内工具下拉（含分享等）、页面级反馈（Turbo Stream 提交）、首页反馈横幅；反馈辅助提示语支持 JSON 配置
- **栏目与首页联动**：页面可设为栏目页展示子页面列表；可勾选「显示在首页快速指引」出现在默认首页入门区块
- **七种界面语言**：`zh-CN`、`zh-TW`、`en`、`ko`、`ja`、`de`、`fr`（前台 `*.json` + 编辑器 `*.schema.json`）
- **品牌与扩展**：Logo、公司名称、版权/备案信息；`<head>`、顶栏、页脚自定义 HTML；微信公众号分享（AppID / Secret）
- **现代前端栈**：Tailwind CSS 4、daisyUI、Stimulus、Alpine.js、Turbo；资源由 `src/` 构建至 `assets/`
- **主题编辑器预览**：按语言提供 Class经典布局 / Card文档布局 / Tree文档布局 首页、正文页及 AI 侧栏截图

### 模板类型一览

| 模板文件 | 类型 | 说明 |
| --- | --- | --- |
| `index.liquid` | 首页 | Class经典布局：柔色渐变 Hero、频道入门卡片、推荐阅读 |
| `index.tree.liquid` | 首页 | Tree文档布局：主标题、快速指引卡片、推荐列表、底部求助/反馈 |
| `index.card.liquid` | 首页 | Card文档布局：左栏 Logo + 搜索 + 分组导航；眉题、Copy page、页内目录 |
| `page.liquid` | 内容页 | 文档正文；布局与首页变体一致；支持栏目页、标签、图标等 |
| `search.liquid` | 搜索 | 预留 |
| `feedback_*_turbo_stream.liquid` | 反馈流 | 页面反馈提交的 Turbo Stream 响应 |

### 全局配置分组（主题设置）

| 分组 | 主要配置项 |
| --- | --- |
| **通用设置** | Logo 图片、公司名称、备案/版权信息、AI 回答结束语、页面反馈辅助提示语 |
| **页面配置** | 显示发布时间、显示作者、是否启用页面反馈、显示页面工具下拉菜单 |
| **微信配置** | 开发者 ID / 密码、是否启用微信分享、调试模式 |
| **自定义 HTML 代码** | `<head>` 注入、顶部导航区域、页脚区域、页脚内菜单区域 |

### 首页模板配置要点（各变体共有）

| 配置组 / 项 | 说明 |
| --- | --- |
| **首页配置** | `home_page_path`：站内页面路径，访问 `/` 时展示该页；留空则启用变体默认首页 |
| **首页内容** | 主标题、副标题/导语、热门搜索（换行多个关键词）、入门/指引板块、推荐阅读、底部求助文案及显示开关 |
| **布局与导航** | 页面宽度（窄/适中/宽/满屏）、左/右侧边栏宽度、页面左侧自定义链接列表 HTML |

### 内容页模板配置要点

| 配置项 | 说明 |
| --- | --- |
| 是否为栏目页面 | 作为栏目页时展示子页面列表 |
| 显示在首页快速指引 | 出现在默认首页「快速指引 / 入门」区块 |
| 页面图标 | 建议 1:1，用于列表与导航展示 |
| 页面标题 / 内容 | 知识库同步，只读 |
| 页面描述、页面标签 | 可编辑，支持检索与展示 |
